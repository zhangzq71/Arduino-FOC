## Library Examples
Examples folder structure
```shell
│ examples
│   ├─── encoder examples             # EXAMPLES OF ENCODER APPLICATIONS
│   │   ├───angle_control                   # example of angle control loop with configuration
│   │   ├───change_direction                # simple motor changing velocity direction in real time
│   │   ├───encoder_example                 # simple example of encoder usage
│   │   ├───find_pole_pairs_number          # simple code example estimating pole pair number of the motor
│   │   ├───HMBGC_example                   # example of code to be used with HMBGC controller with
│   │   ├───velocity_control                # example of velocity control loop with configuration
│   │   ├───voltage_control                 # example of the voltage control loop with configuration
│   │   └───velocity_PI_tuning              # code example of tuning velocity controller by using serial terminal
│   │
│   └─── magnetic sensor examples      # EXAMPLES OF MAGNETIC SENSOR APPLICATIONS AS5047/48
│       ├───angle_control                   # example of angle control loop with configuration
│       ├───change_direction                # simple motor changing velocity direction in real time
│       ├───find_pole_pairs_number          # simple code example estimating pole pair number of the motor
│       ├───magnetic_sensor_only_example    # simple example of magnetic sensor usage
│       ├───velocity_control                # example of velocity control loop with configuration
│       ├───voltage_control                 # example of the voltage control loop with configuration
│       └───velocity_PI_tuning              # code example of tuning velocity controller by using serial terminal
```
